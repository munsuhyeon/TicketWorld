# SpringBoot 공연예매사이트 - TicketWorld  
+ 원하는 공연을 선택해서 예매할 수 있는 웹사이트<br/>
+ 회원가입,로그인,게시판,예매,취소 다양한 기능 사용가능<br/>
+ 사용자페이지,관리자페이지로 나뉘어짐
